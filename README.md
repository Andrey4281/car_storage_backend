# Cars storage
<p>It is internet shop for selling cars. You can see backend's part in this repository. 
Backend part interact with frontend part by REST API. You can see code of frontend part in the next repository: https://github.com/Andrey4281/car_storage_frontend.
In this project were the next technologies used: Spring Boot 2.0, Angular 8, Spring JPA, Spring Security (JWT), Spring REST.</p><br>
